LOVERS CALENDAR 0.8.7

ТУТОР НА РУССКОМ

ТВОИ ДЕЙСТВИЯ:
1. Создай пустой репозиторий на GitHub.
2. Открой файл KEY_API.
3. Вставь туда ссылку репозитория в строку GITHUB_REPOSITORY_URL.
4. Проверь строку GITHUB_USERNAME.
5. Создай GitHub PAT.
6. Очень важно: у токена должны быть права не только на репозиторий, но и на workflow.
7. Запусти файл GITHUB\PUSH_TO_GITHUB.bat.
8. Дождись завершения push.
9. Автоматически открой Actions в своем репозитории.
10. Открой сборку ios-cloud-build.
11. Скачай артефакт iOS Simulator.
12. Загрузи его в Appetize и смотри результат в браузере.

ЕСЛИ ВИДИШЬ ОШИБКУ ПРО workflow scope
Это значит, что твой PAT создан без права менять .github/workflows.
Нужно создать новый токен и добавить:
- repo
- workflow

ЧТО НЕ НУЖНО ДЕЛАТЬ
- не нужно использовать browser login Git Credential Manager
- не нужно вставлять git@github.com SSH ссылку
- не нужно добавлять KEY_API в git
