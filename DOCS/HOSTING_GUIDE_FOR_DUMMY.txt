﻿КАК ПОДГОТОВИТЬ ВЕРСИЮ ДЛЯ СЕРВЕРА

Теперь серверный маршрут лежит в папке SERVER.

1. Открой KEY_API
2. Если облако пока не нужно — оставь VITE_SUPABASE_URL и VITE_SUPABASE_ANON_KEY пустыми
3. Запусти SERVER\BUILD_SERVER_PACKAGE.bat
4. Появится папка SERVER\READY
5. Там будет zip и распакованная серверная папка

Что можно сделать дальше:
- загрузить распакованную папку на VPS
- или загрузить zip на сервер и распаковать его там
- после загрузки запустить AUTO_UNPACK_AND_START.sh

Если нужен настоящий облачный режим:
- создай проект облака
- выполни SQL из SUPABASE_SQL_SETUP.sql
- пропиши ключи в KEY_API
- заново собери серверный пакет

Важно:
- server.js — это простой Node-сервер для статики и SPA fallback
- без ключей приложение просто остаётся локальным
- Android-маршрут при этом всё равно может ставить apk на телефон локально
