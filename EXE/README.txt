﻿EXE
- Запусти BUILD_EXE.bat
- Готовые exe-файлы появятся в EXE\READY
- Ключи и пути теперь хранятся в KEY_API в корне проекта

Сетевой момент:
- Интернет нужен только для первого npm install.
- Сам календарь может работать локально без сети.
- Облачная синхронизация вернётся после появления сети, если ключи заполнены в KEY_API.
