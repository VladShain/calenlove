0.9.7
- iOS GitHub Actions simulator build больше не идет через `tauri ios build`, который уводил сборку в device / signing маршрут и требовал development team
- CI маршрут переведен на прямой unsigned xcodebuild для `iphonesimulator` с отключенной подписью: CODE_SIGNING_ALLOWED=NO, CODE_SIGNING_REQUIRED=NO

- поиск собранного `.app` переведен на DerivedData после unsigned simulator build
- версия проекта обновлена до 0.9.7
